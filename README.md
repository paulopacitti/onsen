- **Water sounds** straight in your **terminal** with this beautiful `cli`!
- Tested in **Linux** 🐧 and **macOS** 🍎
- _Tattoo-friendly_ 👍

## Installation ⬇️
`pip3 install onsen`

## Usage ⌨️
- Type `onsen` **anywhere** in your terminal :sparkles:
